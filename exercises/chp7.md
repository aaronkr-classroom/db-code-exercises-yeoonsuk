# 7장 연습문제

## 개념퍼즐

**가로**

1. 행과 열로 구성된 2차원 테이블 구조 <--- 릴레이션
2. DB를 2치원 테이블， 즉 랄레이선의 집합으로 모델링하는 논리적 데이터 구조 <---- 관계데이터모델 
3. 논리적으로 더 이상 분해할 수 없는 값 <------- 원자값
4. 후보키 가운데 업무에 적합하게 선택된 하나의 키 <-------- 기본키
5. 하나의 랄레이션이 갖는 속성의 수 <---------- 차수
7. DB어| 저장된 데이터의 정확성과 일관성을 유지하기 위해 DB가 항상 만족해야 하는 조건 <--- 무결성제약조건 
8. 릴레이션 간의 관계를 정의하는 키 <-- 외래키
10. 랄레이션의 각 행을 의미하며， 개체 인스턴스에 대응히는 것 <--- 투플 
11. 유일성은 만족하지만 최소성을 만족하지 않는 속성 집합 <---- 슈퍼키
12. 하나의 속성이 취할 수 있는 모든 값의 집합  <-------- 도메인

**세로**

1. 럴레이션을 구성하는 속성들의 집합 <--- 릴레이션스키마
4. 하나의 렬레이션이 갖는 투플의 수 <---- 기수
6. 키 가운데 유일성과 최소성을 모두 만족하는 속성 집합 <----- 후보키
9. 기본키를 제외한 나머지 후보커들 <----- 대체키

## 연습문제

1. 다음중관계 데이터 모넬에 대한설명이 잘못된 것은? 
  1. DB를 릴레이션의 집함으로 모델링히는 논리적 데이터 구조다 
  2. 릴레이션은매우구체적인 개념이다. <------------------------------
  3. 개체와 관계가 모두 2치원 테이블， 즉 렬레이션으로 표현된다. 
  4. ER 모벨을 관계 데이터 모델로 변환하는 것이 매우 쉽다. 

2. 디음 중 릴레이션 개념과 일반적 인 테이블 개념의 차이점을 바르게 설명한 것은? 
  1. 하나의 릴레이션은 여러 형태의 테이블로 표현될 수 있다. <-------------------
  2. 하나의 릴레이션은 한 가지 형태의 테이블로 표현될 수 있다. 
  3. 대이블은중복되는행을허용하지 않는다. 
  4. 릴레이션은 중복되는 행을 허용한다 

3. 관계 데이터 모델과 관련된 용어와 설명을 바르게 연결하시오. 
  1. 투플  - b
  2. 기수 - d
  3. 차수 - a
  4. 도메인 - c
  - a. 하나의 릴레이션이 갖는 속성의 수 
  - b. 릴레이션의 각 행row을 의미함 
  - c. 하나의 속성이 취할 수 있는 모든 값의 집합 
  - d. 하나의 릴레이션이 갖는 투플의 수 

4. 다음 중 릴례이션 스카마와 릴레이션 인스턴스의 차이점을 잘못 설명한 것은? 
  1. 릴레이션 스키마는 릴레이션을 구성하는 속성들의 집합이다. 
  2. 릴레이션 인스턴스는 릴레이션을 구성히는 투플들의 집합이다. 
  3. 릴레이션 스카마는 시간이 흘러도 변하지 않는 정적인 성질을 갖는다 
  4. 릴레이션 인스턴스는 시간이 흘러도 변하지 않는 정적인 성질을 갖는다. <---------------------
