## 연습문제

1. 디음 중 이상 현상에 대한 설명이 잘못된 것은?
  - 이상 현상은 관계 DB를 조작할 때 데이터의 중복으로 인해 발생하는 문제를 의미한다.
  - 삽입 이상은 어떤 데이터를 삽입하면 불필요한 데이터도 함께 삽입되는 현상이다. <--------------------------
  - 삭제 이상은 어떤 데이터를 삭제하면 유용한 데이터도 함께 삭제되는 현상이다.
  - 갱신 이상은 중복된 데이터 가운데 일부만 갱신되어 데이터의 불일치가 발생하는 현상이다.

2. 이상현상의 원인과해결 방법에 대한설명이 잘못된 것은?
  - 이상 현상의 근본 원인은 성능 향상을 위해 데이터가 중복되었기 때문이다. <------------------------------------
  - 이상 현상의 근본 원인은 속성 간에 존재하는 여러 개의 종속 관계를 하나의 릴레이션으로 표현했기 때문이다.
  - 이상 현상을 해결하려면 속성 간의 여러 종속 관계를 분해해야 한다.
  - 이상 현상을 해결하려면 하나의 종속 관계를 하나의 릴레이션으로 표현해야 한다.

3. 함수적 종속에 대한설명이 잘못된 것은?
  - 함수적 종속은 릴레이션이 갖고 있는 이상 현상을 발견하기 위해 속성 간의 종속 관계를 규명하는 개념이다.
  - 속성 X의 각 값에 대해 속성 Y의 값이 오직 하나만 연관되면 속성 X는 속성 Y에 함수적으로 종속된다고 한다. <----------------------------
  - 함수적 종속을 X • Y로 표현했을 때 X를 결정자， Y를 종속자라고 부른다.
  - 한 릴레이션에 속한 속성 간의 복잡한 함수적 종속 관계를 쉽게 이해할 수 있도록 그림으로 표현한 것이 함수적 종속 다이어그램이다.

4. (________) 안에 적합한 용어는?

> 결정자인 복합키에 포함되는 속성 가운데 단독으로 결정자 역할을 하는 속성이 없으면， 종속자가 결정자에 (________)이된다.
정답: 완전 함수적 종속

5. (________) 안에 적합한 용어는?

> (________)이란 정규호를 통해서 랄레이션을 분해했을 때 손실되는 정보가 없어야 한다는 것이다.
정답: 정보의 무손실성

6. ‘도서맹묘잎， 제목， 발행일， 저재id， 저지국적)’ 릴레이션의 함수적 종속 관계가 다음과 같이 표현되었다. 이 릴레이션이 만족히는 정규형은 무엇인가? 답에 대한 이유도 설명하시오.

정답:
부분 함수적 종속이 존재하지 않으므로 제2 정규형을 만족하고, 이행적 함수적 종속
(ISBN → 저자id, 저자id → 저자국적)이 존재하므로 제3 정규형은 만족하지 않음

7. 역정규화가 필요한 이유에 대해 설명하시오.

정답:
정규화는 불필요한 데이터의 중복은 감소하지만 질의에 대한 응답시간이 증가해서 성능
저하를 초래할 수 있다. 따라서 성능이 중요한 경우 데이터 중복과 이상 현상을 대가로
치르면서 보다 낮은 정규형으로 되돌아가는 역정규화가 필요하다.

8. 역정규화에 대한설명으로잘못된 것은?
  - 응답시간이 감소한다.
  - 갱신 이상이 발생할 수 있다.
  - 릴레이션 수를 줄이기 위해 이미 분해된 둘 이상의 릴레이션을 통합히는 작업이다. <----------------------------
  - 무결성 보장을 위한 비용이 증가한다.
